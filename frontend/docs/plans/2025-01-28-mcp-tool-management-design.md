# MCP 工具管理页面设计

**日期**: 2025-01-28
**目标用户**: 普通业务用户

---

## 一、功能概述

为 MCP 服务器管理提供独立的前端页面，支持：

1. 查看和管理 MCP 服务器及其工具
2. 工具级别的启用/禁用控制
3. 添加自定义工具到服务器
4. AI 辅助生成工具定义
5. 显示工具定义的 Token 上下文占用

### AI 生成工具定义

用户输入自然语言描述，AI 自动生成 JSON Schema：

```
输入: "查询 GitHub 仓库的 Issue 列表，支持按状态筛选"

AI 生成:
{
  "type": "object",
  "properties": {
    "owner": { "type": "string", "description": "仓库所有者" },
    "repo": { "type": "string", "description": "仓库名称" },
    "state": {
      "type": "string",
      "enum": ["open", "closed", "all"],
      "description": "Issue 状态"
    }
  },
  "required": ["owner", "repo"]
}
```

## 五、Token 占用计算

### 计算逻辑

将工具的完整定义（name + description + inputSchema）转换为字符串，使用 tokenizer 计算 token 数量。

```
工具定义字符串长度 → TikToken → Token 数量
```

### 显示规则

| Token 范围 | 颜色    | 标识     |
| ---------- | ------- | -------- |
| < 500      | 🟢 绿色 | 低占用   |
| 500-1500   | 🟡 黄色 | 中等占用 |
| > 1500     | 🔴 红色 | 高占用   |

### 后端 API

```python
GET /api/v1/mcp/servers/{id}/tools          # 获取工具列表（含 token 占用）
GET /api/v1/mcp/servers/{id}/token-count    # 获取总 token 占用
PUT /api/v1/mcp/servers/{id}/tools/{tool_id}/enabled  # 切换工具启用状态
POST /api/v1/mcp/servers/{id}/tools          # 添加自定义工具
POST /api/v1/mcp/tools/ai-generate           # AI 生成工具定义
```

## 七、UI 组件复用

使用现有的 shadcn/ui 组件：

- `Card`, `CardHeader`, `CardContent` - 服务器卡片
- `Drawer` 或 `Sheet` - 侧边抽屉
- `Dialog` - 添加工具对话框
- `Switch` - 工具启用/禁用开关
- `Badge` - 状态显示
- `Input` - 搜索和表单输入
- `Button` - 操作按钮
