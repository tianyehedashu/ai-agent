- **统一技能契约**：根目录 `.agents/skills` 定义了跨模块的通用开发标准（如 React 组合模式、数据库 Schema 设计、K8s 调试），作为前后端及运维团队的共同知识底座。
- **自动化质量门禁**：通过 `.claude/agents` 和 `.github/workflows` 将代码规则检查、Pylint 修复及 Sonar 扫描嵌入 CI/CD 流程，确保后端 Python 与前端 TypeScript 代码在合并前符合既定架构规范。
- **配置与环境隔离**：利用根级 `.env` 系列文件与 `backend/config` 中的 TOML 环境配置，结合 `deploy` 模块的 K8s/Docker 编排，实现开发、测试、生产环境的标准化隔离与一键式部署。
- **文档驱动开发**：`.qoder/repowiki` 作为中央知识库，同步映射后端网关、前端应用及部署架构的设计决策，确保工程演进过程中的信息一致性与可追溯性。
