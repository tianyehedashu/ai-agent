- **技术栈**：React 18 + TypeScript + Vite + Tailwind CSS + Radix UI。
- **状态管理**：Zustand 处理全局状态（Auth, Chat, Sidebar），TanStack Query (React Query) 负责服务端数据缓存与同步。
- **路由架构**：React Router v6 实现嵌套路由，`/gateway/*` 为独立的管理控制台子树，支持团队上下文切换。
- **API 层**：封装 `ApiClient` 统一处理 JWT/SSO 认证、自动 Token 刷新及 SSE 流式响应；按业务域（Gateway, Chat, Admin）组织 API 模块。
- **组件组织**：采用 Feature-Sliced 风格，`src/features` 存放高内聚的业务模块（如 `gateway-credentials`, `gateway-models`），`src/components/ui` 为原子化基础组件。
- **构建优化**：Vite 配置代理解决开发环境 CORS，生产环境通过 `VITE_APP_ROOT` 支持子路径部署。
