- **环境启动**: 使用 `make dev` 或 `uv run python scripts/run_server.py` 启动，脚本会自动处理 Windows 下的 `WindowsSelectorEventLoopPolicy` 以兼容 `asyncpg`。
- **数据库迁移**: 开发环境使用 `make db-upgrade` (Alembic Python 脚本)，生产环境运维需手动执行 `alembic/sql/` 目录下的纯 SQL 迁移文件。
- **测试执行**: 使用 `make test` 并发运行单元测试与集成测试，E2E 测试需预先启动 Docker 基础服务并执行 `make test-e2e`。
