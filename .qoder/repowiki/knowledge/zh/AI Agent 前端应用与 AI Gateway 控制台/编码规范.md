- API 客户端统一使用 `apiClient` 实例，通过 `getAuthToken()` 从 Zustand Store 获取 Token，避免直接操作 localStorage。
- TanStack Query 的 `queryKey` 定义为纯函数并集中导出（如 `query-keys.ts`），遵循 `[domain, resource, id, ...params]` 的数组结构以支持精确失效。
- 业务功能模块（Features）内部遵循 UI 组件、自定义 Hooks 与工具函数分离的原则，并通过 `index.ts` 或特定入口文件暴露公共接口。
- SSE 流式请求在 `apiClient.stream` 中统一处理，支持 AbortSignal 取消、自动 Token 续期及网络错误语义化转换。
