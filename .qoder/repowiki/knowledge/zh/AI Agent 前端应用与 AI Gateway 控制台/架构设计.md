- **技术栈核心**：采用 Vite + React 18 + TypeScript 构建，使用 Tailwind CSS 进行样式设计，Radix UI 提供无头组件基础。
- **状态与数据流**：使用 Zustand (persist middleware) 管理全局认证与会话状态（`src/stores/auth.ts`），通过 `apiClient` (`src/api/client.ts`) 统一处理 JWT/SSO 鉴权、Token 自动续期及 SSE 流式请求；业务数据层基于 TanStack Query 实现服务端状态缓存与同步。
- **路由与布局**：基于 React Router v6 实现嵌套路由，`AppLayoutRoutes` (`src/routes/app-layout-routes.tsx`) 定义主应用结构，包含 Chat、Studio、Admin 及 Gateway 子模块；Gateway 模块采用团队上下文驱动的路由设计 (`/gateway/teams/:teamId/*`)。
- **模块化组织**：采用 Feature-Sliced 风格组织代码，`src/features` 下按业务域（如 `gateway-models`, `gateway-credentials`）聚合 UI、Hooks 与逻辑；`src/components/ui` 封装通用原子组件；`src/api/gateway` 提供领域特定的 API 客户端封装。
