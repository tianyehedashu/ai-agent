- 采用 `cn(...)` 工具函数合并 Tailwind 类名，利用 `clsx` 和 `tailwind-merge` 解决样式冲突。
- API 请求统一通过 `apiClient` 实例发起，支持自动 Token 续期（Refresh Token）及 SSO Cookie 透传。
- 业务逻辑封装为自定义 Hooks（如 `useGatewayModelMutations`），将数据获取、变更及副作用管理与 UI 组件解耦。
- 使用 `queryKey` 工厂函数（如 `gatewayModelsListQueryKey`）规范化管理 React Query 缓存键，确保缓存失效的精确性。
- UI 组件广泛使用 `forwardRef` 并配合 `displayName`，确保与 Radix UI 等库的兼容性及调试便利性。
