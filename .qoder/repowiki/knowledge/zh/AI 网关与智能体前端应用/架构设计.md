- **技术栈与构建**：采用 Vite + TypeScript + React 18 架构，使用 Tailwind CSS 进行样式管理，通过 `lazyWithReload` 实现路由级代码分割与部署后自动刷新。
- **状态与数据流**：以 TanStack Query (React Query) 为核心处理服务端状态（API 缓存、同步），Zustand 管理全局客户端状态（认证 Token、团队上下文、聊天会话）。
- **模块化设计**：遵循 Feature-Sliced 思想，`src/features` 按业务域（如 `gateway-models`, `gateway-credentials`）组织，内部包含 hooks、utils 及 UI 组件；`src/api/gateway` 提供统一的 API 客户端抽象。
- **权限与路由**：通过 `AuthProvider` 实现混合认证（Local JWT / SSO Cookie），结合 `useGatewayPermission` 钩子实现细粒度的 RBAC 权限控制；路由配置在 `app-layout-routes.tsx` 中集中管理，支持嵌套布局。
- **UI 基础层**：基于 Radix UI  primitives 封装无头组件，配合 `class-variance-authority` 实现可复用的 UI 系统（位于 `src/components/ui`）。
