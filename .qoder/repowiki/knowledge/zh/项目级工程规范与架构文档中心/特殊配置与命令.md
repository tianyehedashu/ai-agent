- **生产发版流程**：通过 `ssh wuhan-ali` 登录运维节点，使用 `kubectl set image` 更新镜像并执行 `alembic upgrade head` 完成数据库迁移。
- **单机部署命令**：支持通过 `make deploy`（标准）、`make deploy-quick`（快速重启）及 `make deploy-rebuild`（强制重建）等 Makefile 目标管理 Docker Compose 环境。
- **质量门禁扫描**：通过 `make sonar` 或 GitHub Actions 触发 SonarCloud 扫描，依赖 `SONAR_TOKEN` 环境变量进行身份验证。
