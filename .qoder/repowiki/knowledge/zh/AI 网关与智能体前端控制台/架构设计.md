- **技术栈**: React 18 + TypeScript, Vite 构建, React Router v6 (扁平路由), TanStack Query (服务端状态), Zustand (客户端状态)。
- **分层架构**: `src/api` 封装 HTTP 客户端与 SSE 流；`src/features` 按业务域（如 gateway-credentials, gateway-models）组织可复用逻辑与 UI；`src/pages` 组合特性模块构成页面；`src/components/ui` 基于 Radix UI 和 Tailwind CSS 的基础组件库。
- **状态管理**: 认证状态由 `authStore` (Zustand + localStorage 持久化) 统一管理，API 客户端从中获取 Token；服务端数据通过 React Query 缓存与同步。
- **路由设计**: 采用扁平化路由配置，`/gateway/*` 下通过团队 ID (`teamId`) 实现多租户上下文隔离，支持个人与团队资源的统一视图切换。
