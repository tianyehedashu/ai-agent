- 模块由独立的功能脚本组成，无内部包依赖，通过文件系统路径（如 `backend/`, `frontend/`）与项目其他部分交互。
- 采用跨平台对等设计，核心功能（Sonar 扫描、SonarCloud API 交互）均提供 Bash (`.sh`) 和 PowerShell (`.ps1`) 两种实现，以适配 Linux/macOS 和 Windows 环境。
- `sonarcloud_api.py` 作为独立的 Python CLI 工具，封装了 SonarCloud REST API 的复杂交互（分页获取 Issues、Metrics），并提供 JSON/CSV/HTML 多格式报告渲染逻辑，弥补了 Shell 脚本在复杂数据处理上的不足。
- `run-e2e.ps1` 充当 E2E 测试的编排入口，负责基础设施（Docker 容器）、数据库迁移（Alembic）、应用启动（Uvicorn）及测试执行（Pytest）的生命周期管理。
