- **统一编排层**：根目录 `Makefile` 作为核心入口，代理并串联 `backend` (uv/Make) 与 `frontend` (npm) 的子任务，实现一键式依赖安装、开发与检查。
- **基础设施共享**：`docker-compose.yml` 定义了 PostgreSQL、Redis、Qdrant 等共享中间件，为前后端提供统一的本地开发环境与数据持久化卷。
- **跨模块质量治理**：根级 `sonar-project.properties` 聚合了 Python 与 TypeScript 的源码路径、测试报告及覆盖率指标，配合 `.github/workflows` 实现全栈代码质量的统一扫描与门禁控制。
- **标准化部署流**：通过 `deploy/` 目录下的脚本与远程 PowerShell 工具，将多环境（K8s/HiGress）配置与远程同步逻辑抽象为统一的 `make deploy` 工作流。
